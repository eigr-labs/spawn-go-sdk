# [Spawn](https://github.com/eigr/spawn)

**Spawn Actor Mesh Runtime for Go**

## **Installation**

### Install Spawn CLI

```SH
curl -sSL https://github.com/eigr/spawn/releases/download/v1.4.3/install.sh | sh
```

# **Getting Started**

Install go package.

```go
```

### Create a new project with

```SH
spawn new go hello_world
```
